# Workouts!!

My attempt at making a gtk/gnome/libadwaita workout app since i couldn't seem to find anywhere.

## Requirements

- Python 3.11+
- PyGObject

I got these on fedora like this. If this project grows ill update this page

sudo dnf install -y python3-gobject gtk4 libadwaita

## Run

python3 -m gnome_workouts
